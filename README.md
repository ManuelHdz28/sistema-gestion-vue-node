# Sistema de Gestión de Usuarios

Proyecto full stack desarrollado con Vue 3, Node.js y PostgreSQL.

## Tecnologías

- Vue 3
- Node.js + Express
- PostgreSQL
- Docker (para base de datos)

## Funcionalidades

- Registro de usuarios
- Listado de usuarios
- Activación / desactivación de usuarios

## Objetivo

Proyecto práctico para demostrar conocimientos en desarrollo de sistemas web.
